## REST API for a library - java bootcamp (zadanie 17.7)

Requirements are set to handle localhost SQL database, allowing to:
- add user,
- add book,
- add an exemplar,
- change an exemplar status,
- get available exemplars list,
- rent an exemplar,
- return a book
